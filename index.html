Команды Telegram Бота

Основные команды

  • /start - Запуск бота
  • /faq - Часто задаваемые вопросы
  • /addtask - Добавить задачу
  • /listtasks - Список задач
  • /removetask - Удалить задачу
  • /quiz - Викторина
  • /ban - Забанить пользователя
  • /mute - Замутить пользователя
  • /unmute - Размутить пользователя
